**The TL;DR:**
LinkedIn no longer provides public access to its APIs. Since 2015, all LinkedIn API access requires joining the LinkedIn Partner Program. Individual devs can't just create an app and start hitting endpoints.

**What's available:**

| Access Tier | What You Get | Requirements |
|-------------|--------------|--------------|
| **Self-Service (Open)** | Sign in with LinkedIn, basic profile access, share content | Any dev, via Developer Portal |
| **Marketing Developer Platform** | Ads API, Campaign Management, Community Management | Application + approval, proven marketing platform |
| **Sales Navigator** | Lead management, sales data | Enterprise partnership |
| **Talent Solutions** | Recruiter APIs, job posting, ATS integration | Established HR platform |

**If you just want basic stuff:**
Open Permissions are the only permissions that are available to all developers without special approval. This covers:
- Sign In with LinkedIn (OAuth 2.0 / OpenID Connect)
- Share on LinkedIn
- Basic profile info (with user consent)

- Developer Portal: https://developer.linkedin.com
- Python client: `linkedin-api-python-client` on GitHub
- JS client: `linkedin-api-js-client`
