```python
import time
from picamera import PiCamera
from gpiozero import Button
from PIL import Image
import cups

# Initialize camera
camera = PiCamera()
camera.resolution = (1024, 768)
camera.start_preview(alpha=200)

# Initialize button
button = Button(17)  # GPIO pin 17

# Initialize printer
conn = cups.Connection()
printers = conn.getPrinters()
printer_name = list(printers.keys())[0]  # Use the first available printer

# Function to capture and save photo
def capture_photo():
    timestamp = time.strftime("%Y%m%d-%H%M%S")
    file_path = f"/home/pi/photos/photo_{timestamp}.jpg"
    camera.capture(file_path)
    print(f"Photo saved: {file_path}")
    return file_path

# Function to print photo
def print_photo(file_path):
    print(f"Printing photo: {file_path}")
    conn.printFile(printer_name, file_path, "Photo Booth Picture", {})

# Main loop
try:
    print("Photo booth ready. Press the button to take a picture!")
    while True:
        button.wait_for_press()
        print("Button pressed! Taking picture...")
        
        # Countdown
        for i in range(3, 0, -1):
            print(f"{i}...")
            time.sleep(1)
        
        # Capture photo
        file_path = capture_photo()
        
        # Ask if user wants to print
        print("Do you want to print this photo? (yes/no)")
        if input().lower() == 'yes':
            print_photo(file_path)
        
        print("Ready for next photo!")

except KeyboardInterrupt:
    print("Photo booth stopped.")
finally:
    camera.stop_preview()
    camera.close()
```

This Python script provides a basic framework for the automated camera system. Here's a breakdown of its functionality:

1. It uses the PiCamera library to interface with the Raspberry Pi Camera Module.
2. A physical button (connected to GPIO pin 17) triggers the photo capture.
3. Photos are saved with a timestamp in the filename.
4. It includes a simple printing function using the CUPS (Common UNIX Printing System) library.
5. There's a countdown before each photo is taken.
6. After each photo, it asks if the user wants to print the photo.

To use this script, you would need to:

1. Install necessary libraries (picamera, gpiozero, pillow, pycups)
2. Ensure the Raspberry Pi is properly set up with the camera module and a button connected to GPIO pin 17
3. Configure a printer with CUPS on the Raspberry Pi

This script is a starting point and would need further development for a production-ready system. Some areas for improvement could include:

- Adding a graphical user interface using a library like Tkinter or PyQt
- Implementing the LED control for lighting effects
- Adding photo filters or editing capabilities
- Integrating with a touchscreen for user input
