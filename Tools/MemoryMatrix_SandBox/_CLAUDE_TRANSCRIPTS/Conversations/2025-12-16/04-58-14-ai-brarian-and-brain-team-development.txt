```python
class HandoffQueue:
    """
    Ben drops extractions here.
    Dude picks them up on his tick.
    """
    
    def __init__(self):
        self.pending = []
    
    def enqueue(self, nodes: list):
        """Ben calls this after extraction."""
        self.pending.extend(nodes)
    
    def dequeue_batch(self, max_items: int = 10) -> list:
        """Dude calls this on his periodic tick."""
        batch = self.pending[:max_items]
        self.pending = self.pending[max_items:]
        return batch
```

I extract immediately. The Dude files when he gets around to it. The queue bridges our different rhythms.
