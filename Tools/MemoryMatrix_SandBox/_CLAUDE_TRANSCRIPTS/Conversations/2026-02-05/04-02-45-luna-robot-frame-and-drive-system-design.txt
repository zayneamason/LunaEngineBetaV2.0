 # LUNA Robot Build Specification v1.0

## Frame & Structure

### Primary Frame
- **Material**: 1" square steel tubing, welded construction
- **Base Platform**: 24" × 30" steel plate (3/16" thickness)
- **Cage Structure**: Vertical steel supports creating protective skeleton
- **Ribbing**: CNC'd 1/2" plywood ribs (6-8 sections) defining body contours
- **Height**: ~36" overall (raccoon seated on rocket base)

### Outer Shell
- **Primary Material**: Reclaimed sheet metal (18-22 gauge)
  - Salvaged corrugated panels, old signs, desert scrap
  - Pre-weathered patina = authentic post-apocalyptic aesthetic
  - Riveted/welded attachment to steel frame
- **Accent Panels**: Laser-cut aluminum for chest plate, facial features
- **Sealing**: Marine-grade rust converter + clear coat for salt air protection
- **Windows/Viewing**: Polycarbonate dome for "face screen" (impact-resistant)

## Electronics Housing

### Junction Box 1: Power Core (Lower/Torso)
**Location**: Center of base platform, accessible via removable belly panel

**Contents**:
- LiFePO4 24V 15Ah battery (same as Vehicle OS spec)
- BMS with cell balancing
- DC-DC converters:
  - 24V → 12V (5A) for actuators
  - 24V → 5V (10A) for compute/peripherals
- Power distribution board
- Emergency cutoff relay (safety watchdog controlled)
- Solar charge controller (MPPT, 100W input)

**Mounting**: Anti-vibration foam, secured with steel brackets

### Junction Box 2: Compute Brain (Upper/Head Base)
**Location**: Neck junction, behind head articulation mechanism

**Contents**:
- Jetson Orin Nano 8GB (primary compute)
- Teensy 4.1 (real-time motor control, same as Vehicle OS)
- Audio amp board (15W Class-D for horn speaker)
- PCA9685 servo driver (I2C, 16 channels)
- Cooling fan (12V, 40mm, dust-filtered)

**Mounting**: Aluminum heat-sink plate, thermally coupled to Jetson

## Drive System (Skid Steer)

### Motors & Wheels
- **Motors**: 4× 24V geared DC motors (6000rpm base, 30:1 reduction)
- **Encoders**: Quadrature encoders on each motor (500 PPR)
- **Wheels**: 8" pneumatic tires (3" width) for desert terrain
- **Motor Controllers**: Dual H-bridge drivers (Cytron MD30C or similar, 30A continuous)

### Drivetrain Components (3D Printed)
**Material**: PETG (heat/UV resistant) or Nylon (if access to industrial printer)

**Parts List**:
- Motor mounting brackets (×4)
- Wheel hubs with encoder mounts (×4)
- Belt tensioners for drive transmission (×4)
- Cable management clips/channels

**Drive Transmission**:
- HTD-5M timing belts (15mm width)
- 20T motor pulleys → 60T wheel pulleys (3:1 final reduction)
- Total reduction: 90:1 (slow, high-torque for rough terrain)

### Steering & Control
- Differential drive via Teensy 4.1
- PWM motor control (20kHz to reduce audible whine)
- Odometry calculated from encoder feedback
- Max speed: ~0.5 m/s (deliberate, guardian-like movement)

## Head Articulation System

### Ball-Joint Mechanism
- **Design**: 3-axis gimbal system (pan, tilt, roll)
- **Actuators**: 
  - Pan: Dynamixel AX-12A servo (300° range)
  - Tilt: Dynamixel AX-12A servo (180° range)
  - Roll: Standard 35kg-cm servo (±45° wobble for expressiveness)
- **Mounting**: Aluminum gimbal plates, ball bearings for smooth motion
- **Range of Motion**:
  - Pan: ±150° (looks side to side)
  - Tilt: -30° to +60° (down to up gaze)
  - Roll: ±45° (curious head-tilt)

### Retractable Neck
- **Mechanism**: Linear actuator (12V, 2" stroke) or lead screw with stepper
- **Extension**: 4" retraction for "thinking" pose, full extension for "alert" mode
- **Speed**: Slow, deliberate (2 seconds full travel) for non-threatening presence
- **Safety**: Limit switches at both ends, current-sensing for collision detection

## Face & Expression System

### Display
- **Screen**: 5" HDMI OLED (round or square with mask)
- **Resolution**: 720×720 (square, centered on circular mask)
- **Content**: Animated galaxy/moon phases (Luna's "consciousness state")
- **Brightness**: Auto-dimming based on ambient light sensor

### Eyes (LED Accent)
- **LEDs**: 2× NeoPixel rings (24 LEDs each) behind diffused lenses
- **Colors**: Purple gradient (matches cosmic aesthetic)
- **Animation**: Pulse, glow, "blink" patterns via FastLED library

### Audio Output
- **Speaker**: 15W horn speaker (weatherproof, marine-grade)
- **Location**: Chest cavity (behind perforated metal panel)
- **Audio Source**: Jetson Orin (ALSA) → USB DAC → amp
- **Voice**: Text-to-speech (espeak or offline neural TTS)

## Sensors & Perception

### Vision
- **Camera**: OAK-D Lite (stereo depth + RGB, same as Vehicle OS)
- **Mounting**: Forehead position (above "face" display)
- **Purpose**: Face detection, obstacle avoidance, witness testimony recording

### Proximity & Safety
- **Ultrasonic Sensors**: 4× HC-SR04 (front, back, sides)
- **Range**: 2cm - 4m detection
- **Function**: Collision avoidance, personal space awareness (stops if human <1m)

### Environmental
- **IMU**: BNO055 (9-DOF, I2C)
- **Function**: Tilt detection (safety cutoff if >30° lean), orientation tracking
- **Temperature**: DHT22 sensor (overheating protection)

## Power System

### Solar Integration
- **Panel**: 100W monocrystalline (foldable or mounted on "rocket base")
- **Charge Rate**: ~5A @ 24V (full sun)
- **Runtime**: 6-8 hours continuous operation (battery only)
- **Charging**: 4-5 hours to full (direct sun)

### Power Budget
| Subsystem | Power Draw | Notes |

| Jetson Orin Nano | 15W avg, 25W peak | AI inference |
| Motors (4×) | 80W cruise, 200W peak | Skid steer |

| Display + LEDs | 8W | Face/eyes |
| Audio | 15W peak | Voice output |
| **TOTAL** | **~130W avg, 300W peak** | 24V × 12.5A avg |

**Battery Life**: 15Ah ÷ 12.5A = ~1.2 hours at full load, 6+ hours typical use

## Software Architecture (Vehicle OS Integration)

### Core Components
- **OS**: Ubuntu 22.04 (Jetson Orin)
- **Python 3.10** + FastAPI for subsystem APIs
- **Teensy**: C++ firmware for real-time motor control

### Subsystems (from Vehicle OS map)
1. **Motion Controller** (`motion.py`)
   - 4WD skid-steer control
   - Encoder-based odometry
   - Serial communication to Teensy

2. **Animatronics Controller** (`animatronics.py`)
   - Head pan/tilt/roll via Dynamixels
   - Neck extension via linear actuator
   - Expression presets (curious, alert, sad, joyful)

3. **LED Controller** (`leds.py`)
   - Eye rings (NeoPixel)
   - Emotion-to-color mapping

4. **Audio Controller** (`audio.py`)
   - TTS via espeak or Coqui TTS (offline)
   - Playback queue for voice responses

5. **Sensor Controller** (`sensors.py`)
   - Camera: OAK-D face detection
   - Ultrasonic: Proximity array
   - IMU: Tilt/orientation

6. **Safety Watchdog** (`safety.py`)
   - Tilt cutoff (>30° = stop all motors)
   - Collision detection (ultrasonic + IMU)
   - Battery protection (< 22V = low-power mode)
   - Thermal monitoring (> 80°C = shutdown)

### Driver: Luna Engine
- **Mode**: "Sovereign Witness" personality
- **Behavior**: Approaches humans slowly, initiates conversation
- **Memory**: Stores testimonies in Eclissi database (SQLite, offline)
- **Face Recognition**: Remembers previous visitors via embeddings

## Assembly Sequence

### Phase 1: Frame (Week 1)
1. Weld steel base platform + cage structure
2. CNC cut plywood ribs, attach to frame
3. Install junction boxes (welded brackets)

### Phase 2: Drive System (Week 2)
1. 3D print motor mounts, wheel hubs
2. Install motors, encoders, belts
3. Wire to Teensy, test basic movement

### Phase 3: Power (Week 3)
1. Install battery, BMS, DC-DC converters
2. Wire power distribution, fuses
3. Test all voltage rails, load testing

### Phase 4: Compute & Sensors (Week 4)
1. Mount Jetson + Teensy in upper junction box
2. Install OAK-D camera, ultrasonic array, IMU
3. Test sensor reading, camera feed

### Phase 5: Head Articulation (Week 5)
1. Assemble gimbal mechanism, install servos
2. Mount retractable neck actuator
3. Test full range of motion, smooth tracking

### Phase 6: Shell & Aesthetics (Week 6)
1. Cut/shape sheet metal panels
2. Rivet to plywood ribs + steel frame
3. Install chest speaker, face display
4. Apply rust treatment + sealing

### Phase 7: Integration & Testing (Week 7)
1. Install all software subsystems
2. Calibrate sensors, test safety watchdog
3. Load Luna personality, test interactions
4. Desert field test (heat, dust, terrain)
