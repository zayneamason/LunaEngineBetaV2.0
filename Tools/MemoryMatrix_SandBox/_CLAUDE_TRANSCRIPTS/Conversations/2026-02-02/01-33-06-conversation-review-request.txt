**To add a new assertion:**

```python
# validators.py — ~10 lines
def _check_no_emoji_spam(self, ctx: InferenceContext) -> AssertionResult:
    emoji_count = len(re.findall(r'[\U0001F600-\U0001F64F]', ctx.response))
    passed = emoji_count < 5
    return AssertionResult(
        id="V6",
        name="No emoji spam",
        passed=passed,
        severity="low",
        expected="< 5 emojis",
        actual=f"{emoji_count} emojis found"
    )
```

```python
# In Director, at the instrumentation point:
context.new_metric = some_value  # Add to InferenceContext
```

```jsx
// Frontend — add to the grid
```
