## New Component: Session Ring

The missing piece. A circular buffer in Hub that tracks:

```python
@dataclass
class SessionPresence:
    interface: str          # "voice" | "desktop" | "mobile"
    started_at: datetime
    last_active: datetime
    turn_count: int
    summary: str            # Rolling 1-liner of what's happening
    is_active: bool         # Currently in conversation?

class SessionRing:
    """
    Tracks Luna's presence across interfaces.
    Enables cross-interface awareness.
    """
    
    def __init__(self, max_sessions: int = 10):
        self.ring: deque[SessionPresence] = deque(maxlen=max_sessions)
        self.active_session: Optional[SessionPresence] = None
    
    def start_session(self, interface: str) -> SessionPresence:
        """Called when any interface starts a conversation."""
        # If there's an active session on ANOTHER interface, note it
        previous = self.active_session
        
        session = SessionPresence(
            interface=interface,
            started_at=datetime.now(),
            last_active=datetime.now(),
            turn_count=0,
            summary="",
            is_active=True
        )
        
        if previous and previous.interface != interface:
            # HANDOFF MOMENT - Luna is switching interfaces
            previous.is_active = False
            self.ring.append(previous)
        
        self.active_session = session
        return session
    
    def get_recent_presence(self) -> list[SessionPresence]:
        """Get recent sessions for context injection."""
        return list(self.ring) + ([self.active_session] if self.active_session else [])
    
    def format_for_prompt(self) -> str:
        """Format recent presence for system prompt."""
        if not self.ring:
            return ""
        
        lines = ["## Recent Activity"]
        for session in list(self.ring)[-3:]:  # Last 3 sessions
            lines.append(
                f"- {session.interface}: {session.summary} "
                f"({self._time_ago(session.last_active)})"
            )
        return "\n".join(lines)
```

# HANDOFF: Voice → Hub Turn Sync

## Problem

Voice Luna conversations write to local markdown files but never reach Hub's SessionManager. The 60-tick chunking pipeline runs but has nothing to chunk.

## Solution

Add Hub endpoint for turn ingestion. Voice fires async POST after each turn.

---

## Component 1: Hub Endpoint

**File:** `src/hub/api.py`

**Endpoint:** `POST /conversation/turn`

**Request Schema:**
```python
class ConversationTurnRequest(BaseModel):
    role: Literal["user", "assistant"]
    content: str
    interface: Literal["voice", "desktop", "api"]  # future-proof
    session_id: Optional[str] = None  # let Hub manage, or client can specify
    timestamp: Optional[int] = None   # unix ms, defaults to now
    metadata: Optional[dict] = None   # future: emotion, app_context, etc.
```

**Response Schema:**
```python
class ConversationTurnResponse(BaseModel):
    success: bool
    session_id: str
    turn_count: int
    chunking_pending: bool  # True if buffer has unchunked turns
```

**Behavior:**
1. Get `_lunar_behaviour.session_manager`
2. If no active session OR session interface doesn't match → start new session for this interface
3. `session_manager.append_turn(role, content, timestamp)`
4. Return success + session state

**Error Handling:**
- 503 if LunarBehaviour not initialized
- 400 if invalid role/interface

---

## Component 2: Voice Integration

**File:** `src/voice/conversation/manager.py` (preferred) or `src/voice/backend.py`

**Location:** Inside or after `end_turn()`

**Implementation:**
```python
# After existing end_turn logic, fire async POST to Hub
# NON-BLOCKING - do not await, do not let failure stop voice flow

async def _notify_hub_turn(role: str, content: str):
    """Fire-and-forget turn notification to Hub."""
    try:
        async with aiohttp.ClientSession() as session:
            await session.post(
                "http://127.0.0.1:8882/conversation/turn",
                json={
                    "role": role,
                    "content": content,
                    "interface": "voice"
                },
                timeout=aiohttp.ClientTimeout(total=2)
            )
    except Exception as e:
        logger.warning(f"Hub turn notify failed (non-fatal): {e}")

# Call this after end_turn completes:
asyncio.create_task(_notify_hub_turn(role, text))
```

**Key constraints:**
- MUST be non-blocking (fire-and-forget)
- MUST NOT add latency to voice response
- Failure = log warning, continue normally
- 2 second timeout max
