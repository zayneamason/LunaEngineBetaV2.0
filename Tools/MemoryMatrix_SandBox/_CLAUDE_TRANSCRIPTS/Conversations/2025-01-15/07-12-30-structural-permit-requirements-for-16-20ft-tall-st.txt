# Structural Permit Requirements
For structures with heights between 16-20 feet, the following documentation must be submitted for review:

## Required Documentation

### 1. Section 2 Items
All items specified in Section 2 must be included in the submission package. These form the foundational documentation requirements.

### 2. Construction Drawings
Detailed construction drawings must be provided, including:
- Complete dimensional specifications
- Material specifications
- Assembly details
- Foundation plans
- Connection details

### 3. Rigging Documentation
If the structure requires rigging for anchoring:
- Calculation of forces experienced by all anchor wires
- Detailed specifications of wire connection points to the structure
- Wire specifications including material type and load ratings
- Anchoring method details and specifications

### 4. Load Calculations
Comprehensive load analysis must include:

**Wind Load Calculations**
- Wind pressure calculations
- Surface area calculations
- Wind exposure categories
- Design wind speed considerations

**Gravity Load Calculations**
- Structure self-weight calculations
- Live load analysis
- Dead load analysis
- Load combinations

**Dynamic Load Analysis**
- Vibration considerations
- Impact load calculations
- Seismic considerations if applicable

**Calculation Inputs**
- All assumptions used in calculations
- Safety factors applied
- Reference standards used
- Design criteria
