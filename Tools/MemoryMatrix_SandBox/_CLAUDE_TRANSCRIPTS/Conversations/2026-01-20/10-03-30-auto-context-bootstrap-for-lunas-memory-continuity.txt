## The Revolving Window Model

Here's what a real history management system looks like:

### 1. The Three-Tier Buffer

```
┌─────────────────────────────────────┐
│  ACTIVE WINDOW (Always Loaded)     │  ← Last 5-10 turns
│  - Full fidelity                    │  ← Raw conversation
│  - Token-budgeted                   │  ← Guaranteed in context
└─────────────────────────────────────┘
           ↓ (as window slides)
┌─────────────────────────────────────┐
│  RECENT BUFFER (Searchable)        │  ← Last 50-100 turns
│  - Compressed summaries             │  ← Key points extracted
│  - Hybrid searchable (FTS + vector)│  ← Retrievable on demand
└─────────────────────────────────────┘
           ↓ (after session ends)

│  - Chunked & embedded               │  ← Semantic retrieval
│  - Relationship-linked              │  ← Graph-traversable
└─────────────────────────────────────┘
```

### 2. The Windowing Strategy

**Active Window** (Always in context):
- Last 5-10 turns, full text
- Includes current message
- Managed like a ring buffer
- ~500-1000 tokens max

**Recent Buffer** (On-demand retrieval):
- Last 50-100 turns
- Compressed to key points: "User asked about X, Luna explained Y, decision was Z"
- Searchable: "What did we decide about the observability layer?"
- ~2000-3000 tokens if fully loaded

**Archive** (Memory Matrix integration):
- Everything older than Recent Buffer
- Goes through Ben Franklin (extraction) → The Dude (filing)
- Becomes searchable memories with graph relationships
- Infinite retention, smart retrieval

class HistoryManager(Actor):
    """Manages conversation history tiers."""
    
    def __init__(self, hub_api, config):
        self.hub = hub_api
        self.max_active_tokens = config.max_active_tokens
        self.max_recent_age_minutes = config.max_recent_age_minutes
    
    async def tick(self):
        """Check window sizes and trigger rotations if needed."""
        # Check Active Window budget
        active_budget = await self.hub.get_active_token_count()
        if active_budget > self.max_active_tokens:
            await self._rotate_active_to_recent()
        
        # Check for uncompressed Recent turns
        uncompressed = await self.hub.get_uncompressed_recent_turns(limit=1)
        if uncompressed:
            await self.hub.queue_compression(uncompressed[0]['turn_id'])
        
        # Check for archivable Recent turns
        old_recent = await self.hub.get_archivable_turns(
            age_minutes=self.max_recent_age_minutes,
            limit=1
        )
        if old_recent:
            await self.hub.queue_extraction(old_recent[0]['turn_id'])
    
    async def _rotate_active_to_recent(self):
        """Move oldest Active turn to Recent tier."""
        oldest = await self.hub.get_oldest_active_turn()
        if oldest:
            await self.hub.update_turn_tier(
                turn_id=oldest['turn_id'],
                tier='recent'
            )
    
    # Public interface for PersonaCore
    async def get_active_window(self) -> List[ConversationTurn]:
        """Get all Active tier turns for context injection."""
        return await self.hub.get_turns_by_tier(tier='active')
    
    async def search_recent(self, query: str, limit: int = 3):
        """Hybrid search across Recent tier."""
        return await self.hub.search_turns(
