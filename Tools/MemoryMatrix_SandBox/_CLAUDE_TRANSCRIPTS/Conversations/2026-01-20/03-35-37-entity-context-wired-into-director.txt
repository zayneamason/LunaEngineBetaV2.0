1. **Remove entirely** - Pros: Cleaner, no repetitive phrases. Cons: If there's latency before Claude responds, the user sees nothing and might think it's broken. Though streaming should start quickly...

2. **Vary randomly** - Pros: Less repetitive. Cons: Still generic, still doesn't communicate what's actually happening. Random variety can feel even more robotic ("why did the AI suddenly switch how it talks?")

3. **Make contextual** - Pros: Actually communicates what's happening (searching memory vs researching vs thinking). Feels more authentic. Cons: More implementation work, need to maintain the mapping.
