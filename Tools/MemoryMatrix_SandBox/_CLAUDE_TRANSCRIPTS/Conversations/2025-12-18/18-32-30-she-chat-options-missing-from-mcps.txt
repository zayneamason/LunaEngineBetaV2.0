      "args": ["${__dirname}/src/luna_mcp_server/luna_mcp.py"],

        "ANTHROPIC_API_KEY": "${user_config.anthropic_api_key}",
        "LUNA_BASE_PATH": "${__dirname}",
        "HUB_URL": "${user_config.hub_url}"
      }
    }
  },
  "user_config": {
    "anthropic_api_key": {
      "type": "string",
      "title": "Anthropic API Key",

      "type": "string",
      "title": "Hub URL",

**2. Package it:**

```bash
# Install the toolchain
npm install -g @anthropic-ai/mcpb

# In your Luna project root

mcpb pack   # Creates luna-mcp.mcpb
```

## What Luna Would Look Like on Pure API

Your Luna tools (`luna_detect_context`, `memory_matrix_search`, etc.) become **tool definitions** in the API call:

```python
import anthropic

client = anthropic.Anthropic()

tools = [
    {
        "name": "luna_smart_fetch",
        "description": "Fetch relevant memories from Luna's memory graph",
        "input_schema": {
            "type": "object",
            "properties": {
                "query": {"type": "string", "description": "What to search for"},
                "budget_preset": {"type": "string", "enum": ["minimal", "balanced", "rich"]}
            },
            "required": ["query"]
        }
    },
    {
        "name": "memory_matrix_search",
        "description": "Semantic search across Luna's memory nodes",
        "input_schema": {
            "type": "object",
            "properties": {
                "query": {"type": "string"},
            },
            "required": ["query"]
        }
    },
    # ... all your other Luna tools
]

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=4096,
    tools=tools,
    messages=[{"role": "user", "content": "Hey Luna, what do you remember about the BFE?"}]
)

# Handle tool calls yourself
for block in response.content:
    if block.type == "tool_use":
        result = execute_luna_tool(block.name, block.input)  # Your code
        # Send result back to Claude...
```
