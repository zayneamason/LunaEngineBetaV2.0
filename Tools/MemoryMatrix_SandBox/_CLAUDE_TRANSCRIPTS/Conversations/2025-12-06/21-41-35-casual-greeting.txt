**What you just showed me:**

Adjacency matrices in graph theory don't just represent connections—**powers of the matrix reveal path structures**.

- A¹ = direct connections (length 1 paths)
- A² = paths of length 2 (mutual friends, indirect links)
- A³ = paths of length 3 (triangles, clustering)
- A^n = all n-hop paths between any two nodes

**Applied to Luna's memory:**

Instead of choosing "timeline OR graph", you get **both simultaneously through matrix powers**.

```python

# Nodes = memory units (facts, decisions, problems, etc.)
# Edges = relationships (depends_on, enables, contradicts, etc.)

A = adjacency_matrix(memory_graph)

# Query: "What led to this decision?"
# = Find all paths of varying length terminating at decision_node

A^1 → direct dependencies (immediate causes)
A^2 → second-order dependencies (what enabled the enablers)
A^3 → third-order context (the backstory)

# The matrix power IS the temporal depth query
```

**The key insight:**

You don't need separate "linear" and "graph" storage. **The graph structure contains timeline information through path distance**.

- Direct edge = happened in same session or causally adjacent
- 2-hop path = one degree of separation
- 3-hop path = deeper context
- Disconnected subgraphs = unrelated memory clusters

**What this enables:**

1. **Semantic distance queries**: "How related are these two memories?" = shortest path length in A^n
2. **Context expansion**: "Give me everything within 3 hops of this decision" = (A + A² + A³) row/column slice
3. **Clustering detection**: Trace of A³ / 6 = number of triangular relationships (tight concept clusters)
4. **Influence propagation**: "If I change this assumption, what breaks?" = walk forward through A^n
5. **Memory compression**: High connectivity nodes = core concepts, low connectivity = peripheral details

**The timeline lives in edge weights:**

Instead of binary (connected/not), edges can have:
- **Temporal distance** (hours/days between events)
- **Causal strength** (how directly X led to Y)
- **Certainty** (how confident are we in this link)

Then matrix multiplication becomes **weighted path traversal**—favoring recent, strong, certain connections over old, weak, speculative ones.

-- Core memory units
CREATE TABLE memory_nodes (
    id TEXT PRIMARY KEY,  -- uuid
    type TEXT NOT NULL,   -- FACT, DECISION, PROBLEM, ASSUMPTION, etc.
    content TEXT NOT NULL,
    timestamp INTEGER NOT NULL,  -- unix timestamp
    session_id TEXT,
    turn_number INTEGER,
    confidence INTEGER CHECK(confidence >= 0 AND confidence <= 100),
    status TEXT DEFAULT 'ACTIVE',  -- ACTIVE, ARCHIVED, DISPUTED, VERIFIED
    priority TEXT,  -- HIGH, MEDIUM, LOW
    owner TEXT,
    tags TEXT,  -- JSON array as string
    metadata TEXT,  -- JSON blob for flexibility
    created_at INTEGER NOT NULL,
    updated_at INTEGER NOT NULL
);

-- Relationships between nodes
CREATE TABLE memory_edges (
    id TEXT PRIMARY KEY,
    from_node TEXT NOT NULL,
    to_node TEXT NOT NULL,
    edge_type TEXT NOT NULL,  -- depends_on, enables, contradicts, clarifies, related_to
    weight REAL DEFAULT 1.0,  -- causal strength, temporal proximity, etc.
    temporal_distance INTEGER,  -- hours/days between events
    certainty REAL DEFAULT 1.0,  -- 0-1, how confident in this link
    metadata TEXT,  -- JSON blob
    created_at INTEGER NOT NULL,
    FOREIGN KEY(from_node) REFERENCES memory_nodes(id) ON DELETE CASCADE,
    FOREIGN KEY(to_node) REFERENCES memory_nodes(id) ON DELETE CASCADE,
    UNIQUE(from_node, to_node, edge_type)  -- prevent duplicate edges
);

-- Indexes for fast queries
CREATE INDEX idx_nodes_type ON memory_nodes(type);
CREATE INDEX idx_nodes_timestamp ON memory_nodes(timestamp);
CREATE INDEX idx_nodes_status ON memory_nodes(status);
CREATE INDEX idx_nodes_session ON memory_nodes(session_id);
CREATE INDEX idx_edges_from ON memory_edges(from_node);
CREATE INDEX idx_edges_to ON memory_edges(to_node);
CREATE INDEX idx_edges_type ON memory_edges(edge_type);

-- Full-text search for semantic queries
CREATE VIRTUAL TABLE memory_search USING fts5(
    node_id UNINDEXED,
    content,
    tags,
    tokenize='porter'
);

# memory_matrix/core.py

import sqlite3
import networkx as nx
import numpy as np
from typing import List, Dict, Set, Tuple, Optional
from dataclasses import dataclass
from datetime import datetime
import json
import uuid

@dataclass
class MemoryNode:
    id: str
    type: str  # FACT, DECISION, PROBLEM, ASSUMPTION, CONNECTION, ACTION, OUTCOME
    content: str
    timestamp: int
    session_id: Optional[str] = None
    turn_number: Optional[int] = None
    confidence: int = 100
    status: str = "ACTIVE"
    priority: Optional[str] = None
    owner: Optional[str] = None
    tags: List[str] = None
    metadata: Dict = None

@dataclass
class MemoryEdge:
    from_node: str
    to_node: str
    edge_type: str  # depends_on, enables, contradicts, clarifies, related_to
    weight: float = 1.0
    temporal_distance: Optional[int] = None
    certainty: float = 1.0
    metadata: Dict = None

class MemoryMatrix:
    def __init__(self, db_path: str):
        self.db_path = db_path
        self.conn = sqlite3.connect(db_path)
        self.conn.row_factory = sqlite3.Row
        self._init_db()
        self.graph = nx.DiGraph()  # directed graph for causal relationships
        self._load_graph()
    
    def _init_db(self):
        """Initialize database schema if needed"""
        # Execute schema creation SQL from above
        pass
    
    def _load_graph(self):
        """Load graph from database into NetworkX"""
        # Load nodes
        cursor = self.conn.execute("SELECT * FROM memory_nodes WHERE status='ACTIVE'")
        for row in cursor:
            self.graph.add_node(
                row['id'],
                type=row['type'],
                content=row['content'],
                timestamp=row['timestamp'],
                confidence=row['confidence'],
                tags=json.loads(row['tags']) if row['tags'] else []
            )
        
        # Load edges
        cursor = self.conn.execute("SELECT * FROM memory_edges")
        for row in cursor:
            self.graph.add_edge(
                row['from_node'],
                row['to_node'],
                type=row['edge_type'],
                weight=row['weight'],
                certainty=row['certainty']
            )
    
    def add_node(self, node: MemoryNode) -> str:
        """Add memory node to both DB and graph"""
        if not node.id:
            node.id = str(uuid.uuid4())
        
        now = int(datetime.now().timestamp())
        self.conn.execute("""
            INSERT INTO memory_nodes 
            (id, type, content, timestamp, session_id, turn_number, 
             confidence, status, priority, owner, tags, metadata, created_at, updated_at)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
        """, (
            node.id, node.type, node.content, node.timestamp,
            node.session_id, node.turn_number, node.confidence,
            node.status, node.priority, node.owner,
            json.dumps(node.tags) if node.tags else None,
            json.dumps(node.metadata) if node.metadata else None,
            now, now
        ))
        
        # Add to FTS
        self.conn.execute("""
            INSERT INTO memory_search (node_id, content, tags)
            VALUES (?, ?, ?)
        """, (node.id, node.content, json.dumps(node.tags) if node.tags else ""))
        
        self.conn.commit()
        
        # Add to graph
        self.graph.add_node(
            node.id,
            type=node.type,
            content=node.content,
            timestamp=node.timestamp,
            confidence=node.confidence,
            tags=node.tags or []
        )
        
        return node.id
    
    def add_edge(self, edge: MemoryEdge) -> str:
        """Add relationship between nodes"""
        edge_id = str(uuid.uuid4())
        now = int(datetime.now().timestamp())
        
        self.conn.execute("""
            INSERT OR IGNORE INTO memory_edges
            (id, from_node, to_node, edge_type, weight, temporal_distance, certainty, metadata, created_at)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?)
        """, (
            edge_id, edge.from_node, edge.to_node, edge.edge_type,
            edge.weight, edge.temporal_distance, edge.certainty,
            json.dumps(edge.metadata) if edge.metadata else None,
            now
        ))
        self.conn.commit()
        
        # Add to graph
        self.graph.add_edge(
            edge.from_node,
            edge.to_node,
            type=edge.edge_type,
            weight=edge.weight,
            certainty=edge.certainty
        )
        
        return edge_id
    
    def semantic_search(self, query: str, limit: int = 10) -> List[str]:
        """Full-text search for relevant nodes"""
        cursor = self.conn.execute("""
            SELECT node_id, rank
            FROM memory_search
            WHERE memory_search MATCH ?
            ORDER BY rank
            LIMIT ?
        """, (query, limit))
        
        return [row['node_id'] for row in cursor]
    
    def get_context(self, node_ids: List[str], depth: int = 2) -> Set[str]:
        """
        Get all nodes within N hops of anchor nodes.
        This is where the matrix magic happens.
        """
        context = set(node_ids)
        
        for node_id in node_ids:
            # Forward traversal (what this enables)
            for hop in range(1, depth + 1):
                for path in nx.single_source_shortest_path_length(
                    self.graph, node_id, cutoff=hop
                ).keys():
                    context.add(path)
            
            # Backward traversal (what this depends on)
            reverse_graph = self.graph.reverse()
            for hop in range(1, depth + 1):
                for path in nx.single_source_shortest_path_length(
                    reverse_graph, node_id, cutoff=hop
                ).keys():
                    context.add(path)
        
        return context
    
    def smart_fetch(self, query: str, budget_preset: str = "balanced") -> Dict:
        """
        Main retrieval function for Luna.
        Combines semantic search + graph traversal.
        """
        # Map budget to depth
        depth_map = {
            "minimal": 1,   # ~1800 tokens: direct links only
            "balanced": 2,  # ~3800 tokens: 2-hop context
            "rich": 3       # ~7200 tokens: 3-hop deep context
        }
        depth = depth_map.get(budget_preset, 2)
        
        # Find semantically relevant anchor nodes
        anchor_ids = self.semantic_search(query, limit=5)
        
        if not anchor_ids:
            return {"nodes": [], "edges": [], "message": "No relevant memories found"}
        
        # Expand context via graph traversal
        context_ids = self.get_context(anchor_ids, depth=depth)
        
        # Fetch full node data
        placeholders = ','.join('?' * len(context_ids))
        cursor = self.conn.execute(f"""
            SELECT * FROM memory_nodes
            WHERE id IN ({placeholders})
            ORDER BY timestamp DESC
        """, list(context_ids))
        
        nodes = [dict(row) for row in cursor]
        
        # Fetch relevant edges
        cursor = self.conn.execute(f"""
            SELECT * FROM memory_edges
            WHERE from_node IN ({placeholders})
               OR to_node IN ({placeholders})
        """, list(context_ids) * 2)
        
        edges = [dict(row) for row in cursor]
        
        return {
            "nodes": nodes,
            "edges": edges,
            "anchor_ids": anchor_ids,
            "depth": depth,
            "total_nodes": len(nodes)
        }
    
    def trace_dependencies(self, node_id: str, max_depth: int = 5) -> List[List[str]]:
        """Find all dependency paths leading to this node"""
        reverse_graph = self.graph.reverse()
        paths = []
        
        for target in self.graph.nodes():
            if target == node_id:
                continue
            try:
                for path in nx.all_simple_paths(
                    reverse_graph, node_id, target, cutoff=max_depth
                ):
                    paths.append(path)
            except nx.NetworkXNoPath:
                continue
        
        return paths
    
    def find_conflicts(self, node_id: str) -> List[str]:
        """Find nodes that contradict this one"""
        conflicts = []
        for _, target, data in self.graph.edges(node_id, data=True):
            if data.get('type') == 'contradicts':
                conflicts.append(target)
        
        # Also check reverse
        for source, _, data in self.graph.in_edges(node_id, data=True):
            if data.get('type') == 'contradicts':
                conflicts.append(source)
        
        return conflicts
    
    def get_clustering_coefficient(self) -> float:
        """
        Measure of how interconnected the memory graph is.
        High clustering = tight concept relationships.
        """
        return nx.average_clustering(self.graph.to_undirected())
    
    def get_central_nodes(self, limit: int = 10) -> List[Tuple[str, float]]:
        """
        Find most influential memory nodes using PageRank.
        These are core concepts everything else connects to.
        """
        pagerank = nx.pagerank(self.graph)
        sorted_nodes = sorted(pagerank.items(), key=lambda x: x[1], reverse=True)
        return sorted_nodes[:limit]
```

# memory_matrix/extractor.py

import re
from typing import List, Tuple, Optional
from dataclasses import dataclass

@dataclass
class ExtractedRelationship:
    from_content: str
    to_content: str
    edge_type: str
    confidence: float

class ConversationParser:
    """
    Parse conversations to extract memory units and relationships.
    Uses pattern matching + optional LLM assist for complex cases.
    """
    
    # Signal patterns for different node types
    DECISION_PATTERNS = [
        r"(?:we |I )?(?:chose|decided|picked|went with|opted for) (.+?) (?:over|instead of|rather than) (.+?) because (.+)",
        r"(?:the )?decision (?:was )?to (.+)",
        r"(?:we're|we are) going with (.+)"
    ]
    
    PROBLEM_PATTERNS = [
        r"(?:the )?(?:problem|issue|bug) (?:is|was) (.+)",
        r"(?:we're|we are) hitting (.+)",
        r"(.+?) (?:is|was) broken",
        r"(?:need to fix|needs fixing) (.+)"
    ]
    
    ASSUMPTION_PATTERNS = [
        r"(?:assuming|assumes that|assumption:) (.+)",
        r"this (?:depends on|requires|needs) (.+)",
        r"(?:if|when) (.+?), then (.+)"
    ]
    
    OUTCOME_PATTERNS = [
        r"(?:we |I )?(?:reduced|improved|increased|decreased) (.+?) from (.+?) to (.+)",
        r"(?:result|outcome) (?:was|is) (.+)",
        r"(?:now|after that) (.+?) is (.+)"
    ]
    
    # Relationship indicators
    DEPENDENCY_INDICATORS = [
        "depends on", "requires", "needs", "built on", "based on",
        "relies on", "assumes", "if", "when"
    ]
    
    ENABLEMENT_INDICATORS = [
        "enables", "allows", "lets us", "makes it possible",
        "opens up", "unlocks", "leads to"
    ]
    
    CONTRADICTION_INDICATORS = [
        "contradicts", "conflicts with", "opposite of",
        "instead of", "rather than", "but", "however"
    ]
    
    def parse_message(self, message: str, session_id: str, timestamp: int) -> Tuple[List[MemoryNode], List[MemoryEdge]]:
        """
        Extract structured memory from a message.
        Returns (nodes, edges) to be added to the matrix.
        """
        nodes = []
        edges = []
        
        # Split into sentences for atomic extraction
        sentences = re.split(r'[.!?]+', message)
        
        for sent in sentences:
            sent = sent.strip()
            if not sent:
                continue
            
            # Try to classify and extract
            node_type, extracted = self._classify_sentence(sent)
            
            if node_type:
                node = MemoryNode(
                    id=None,  # will be generated
                    type=node_type,
                    content=sent,
                    timestamp=timestamp,
                    session_id=session_id,
                    confidence=70 if node_type != "FACT" else 90  # lower confidence for inferred types
                )
                nodes.append(node)
        
        # Extract relationships between nodes in the same message
        edges.extend(self._extract_relationships(nodes, message))
        
        return nodes, edges
    
    def _classify_sentence(self, sentence: str) -> Tuple[Optional[str], Optional[str]]:
        """Classify sentence type and extract key content"""
        
        for pattern in self.DECISION_PATTERNS:
            match = re.search(pattern, sentence, re.IGNORECASE)
            if match:
                return ("DECISION", match.group(0))
        
        for pattern in self.PROBLEM_PATTERNS:
            match = re.search(pattern, sentence, re.IGNORECASE)
            if match:
                return ("PROBLEM", match.group(0))
        
        for pattern in self.ASSUMPTION_PATTERNS:
            match = re.search(pattern, sentence, re.IGNORECASE)
            if match:
                return ("ASSUMPTION", match.group(0))
        
        for pattern in self.OUTCOME_PATTERNS:
            match = re.search(pattern, sentence, re.IGNORECASE)
            if match:
                return ("OUTCOME", match.group(0))
        
        # Default to FACT if no specific pattern matched
        return ("FACT", sentence)
    
    def _extract_relationships(self, nodes: List[MemoryNode], context: str) -> List[MemoryEdge]:
        """
        Find relationships between extracted nodes.
        Uses keyword matching + proximity heuristics.
        """
        edges = []
        
        for i, node_a in enumerate(nodes):
            for node_b in nodes[i+1:]:
                # Check for dependency indicators
                for indicator in self.DEPENDENCY_INDICATORS:
                    if indicator in context.lower():
                        # Heuristic: node_b depends on node_a if indicator appears between them
                        edges.append(MemoryEdge(
                            from_node=node_a.id,
                            to_node=node_b.id,
                            edge_type="depends_on",
                            certainty=0.6  # auto-extracted = lower certainty
                        ))
                        break
                
                # Check for enablement
                for indicator in self.ENABLEMENT_INDICATORS:
                    if indicator in context.lower():
                        edges.append(MemoryEdge(
                            from_node=node_a.id,
                            to_node=node_b.id,
                            edge_type="enables",
                            certainty=0.6
                        ))
                        break
                
                # Check for contradiction
                for indicator in self.CONTRADICTION_INDICATORS:
                    if indicator in context.lower():
                        edges.append(MemoryEdge(
                            from_node=node_a.id,
                            to_node=node_b.id,
                            edge_type="contradicts",
                            certainty=0.7
                        ))
                        break
        
        return edges
```
