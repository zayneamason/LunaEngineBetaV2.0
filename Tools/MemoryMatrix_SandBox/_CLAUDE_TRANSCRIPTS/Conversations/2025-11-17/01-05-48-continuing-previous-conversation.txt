```bash
# Install Xcode from App Store first (takes a while)
# Then:
npx expo start --ios
```

This opens a virtual iPhone on your Mac.

## Option 3: Android Emulator

**Requires Android Studio (also large):**
- Download Android Studio
- Set up emulator
- Then: `npx expo start --android`

## My Recommendation:

**Start with web version:**
```bash
npx expo start --web
```

**Pros:**
- ✅ Works immediately
- ✅ No extra downloads
- ✅ Fast iteration

**Cons:**
- ❌ Not a "real" mobile app
- ❌ Touch gestures work differently
- ❌ Some mobile-specific features missing

**For testing the UI/flow, web version is perfect.** 

Once `npm install` is done, try:
```bash
npx expo start --web
```

It'll open in Chrome/Safari automatically!
