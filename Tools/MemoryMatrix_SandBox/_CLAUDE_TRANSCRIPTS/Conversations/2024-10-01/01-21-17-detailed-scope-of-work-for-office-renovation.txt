# Detailed Scope of Work

## I. Pre-Construction:
1. Create budget conscious interior design plan
2. Material analysis and selection
3. Budget formulation & finalization
4. Create a construction timeline

## II. Demolition and Preparation:
1. Demo pre-existing interior drywall walls and doors
2. Remove drop ceilings in front main area and back rooms
3. Remove pre-existing flooring in entire space
4. Prep for self-leveling
   a. Remove broken or compromised sheeting as needed
   b. Replace, secure, fasten, strengthen any compromised floor framing
   c. Frame and level as needed
5. Remove old electrical and plumbing runs in ceiling as needed
6. Prepare exposed ceiling members, brick walls for painting

## III. Plumbing and Electrical:
1. Install on-demand water heater and plumbing system
2. Validate existing plumbing and fix any pre-existing plumbing issues
3. Move water and drain lines as needed
4. Validate current electrical system
5. Install meter to sub-panel (by owner)
6. Reroute sub panel main power run in ceiling
7. Re-route any electrical lines from subpanel in ceiling as needed (by owner)
8. Design and implement new electrical schematic
9. Install new power runs from the breaker box to the electrical outlets, switches and junctions as needed per design (by owner)
10. Install house lighting in ceiling throughout per electrical plan
11. Install fire extinguishers and illuminated exit signs for fire safety
12. Move hard wired carbon dioxide/ smoke detectors as needed

## IV. Bathroom Demo and Build-Out:
1. Remove existing tile and vinyl flooring
2. Replace/repair any compromised subfloor sheathing and or structural framing members
3. Repair, replace drywall as needed
4. Level floors and plumb walls as needed
5. Remove existing toilet and dispose
6. Remove existing sink and dispose
7. Remove existing shower and dispose
8. Move the west bathroom wall two feet
9. Build framework for shower floor and curb
10. Pre-plum shower drain connectors
11. Run hot and cold shower lines to shower mixing valve
12. Install shower mixing valve
13. Install water line for shower head
14. Install felt moisture barrier in shower area
15. Install 1/2" green board to shower walls. Silicone seal around the edges
16. Glue and connect shower drain flange to the shower drain line. Secure flange to subfloor
17. Pour deck mud pre-slope to the drain flange level. Pour deck mud to premarked slope lines. Silicone seal edges
18. Install P.V.C. waterproof membrane over pre-slope
19. Install Hardiebacker to shower walls
20. Tape and seal any gaps and screw holes. Silicone seal bottom gaps as needed
21. Pour final deck mud slope. Add metal grid reinforcements
22. Install tile type 1 in bathroom shower area
23. Install tile type 2 in the main bathroom area
24. Fabricate and install metal framed glass shower enclosure with one fixed panel and one swinging glass door with frame
25. Install toilet water lines and angle stop valve
26. Install new toilet and wax ring. Connect plumbing
27. Build and install custom metal vanity cabinet with an in-wall recessed mirror with diffused side lighting
28. Build cast framing and pour cement counter top
29. Sand and polish finish. T.B.D.
30. Seal countertop with food and medical grade clear
31. Run electrical for the fan, lights, outlets and switches
32. Install bathroom fan
33. Install bathroom light
34. Run hot and cold water lines for the sink
35. Install stainless steel sink to cement counter top
36. Install P trap and plumbing components
37. Install faucet and water lines
38. Drywall, taping and mudding as needed
39. Prime and paint walls. Finish paint will be semi gloss
40. Install baseboards

## V. Kitchen Build-Out:
1. Fabricate and install metal and wood cabinets per approved design
2. Fabricate custom cement counter top
3. Mix and pour cement with epoxy additive and colored dye
4. Sand and polish countertop, clean up edges
5. Apply food grade clear coating
6. Install stainless farmhouse style sink and drain
7. Install garbage disposer
8. Install sink and plumbing
9. Install faucet water lines
10. Install LED kitchen light
11. Install 2 GFI 4 output power outlets above counter
12. Install garbage disposal and dishwasher washer unit
13. Install power outlet for the refrigerator/stove
14. Install refrigerator
15. Install power for stove as needed
16. Install stove/oven
17. Install microwave

## VI. Main Room Build-Out:
1. Install drywall on the east wall
2. Patch and paint drywall on east wall
3. Frame and add drywall to interior walls for rooms 1 and 2 per design
4. Patch and paint drywall for rooms 1 and 2 per design
5. Prep and paint ceiling, walls, and exposed brick
6. Add self-leveling as needed throughout space
7. Final painting and touch-ups
8. Fabricate eight 4'x8' architectural movable walls and barn door for rooms 1 and 2
9. Fabricate and install movable wall bracket system
10. Install flooring
11. Install base molding and trim as needed
12. Build out utility closet area on the east wall for washer dryer and storage
13. Build utility counter and shelving
14. Install closet light

## VII. Windows and Doors:
1. Replace all window glass panes with new rated glass panes
2. Prep & paint metal window frames
3. Fix any mechanical issues
4. Repair/ add new security lock to front metal security door
5. Add 3 skylights and 1 fold open skylight (for fire safety exit)
6. Install barn doors & track systems to room 1 & 2
7. Install new swinging door and frame for bathroom/kitchen door access
8. Install new swinging door and frame from room 2 to bathroom
9. In room 2: Install closet security door
10. In room 1: Install door and frame on the east wall
11. Install door trims to all doors and openings
