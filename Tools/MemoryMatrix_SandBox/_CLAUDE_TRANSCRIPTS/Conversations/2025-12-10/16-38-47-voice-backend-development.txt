```python
class TTSProvider(Protocol):
    """Swap implementations freely"""

        """Text → audio bytes"""
        ...
    
    def get_voice_options(self) -> List[VoiceInfo]:
        """Available voices for this provider"""
        ...
    
    def is_available(self) -> bool:
        """Can we use this right now?"""
        ...

# Implementations:
```

Voice Backend calls `tts_provider.synthesize(text)` — doesn't care what's behind it.

**Migration cost with this pattern:** Change one line in config, maybe install a dependency. Done.

**My suggestion:** Start with Apple native. It's already on your machine, latency is instant, quality is fine for development. Get the whole voice pipeline working end-to-end first. Swap TTS when (if) it bothers you.
