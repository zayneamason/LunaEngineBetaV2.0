```python
    strength: float  # 0.0-1.0
    created_at: datetime

@dataclass
class ContextPacket:
    nodes: List[MemoryNode]
    edges: List[Edge]
    anchor_ids: List[str]  # Which nodes matched the search query
    depth: int  # Graph traversal depth used
    total_tokens: int  # Estimated token count
    budget_preset: str
    formatted_markdown: str
```

**Failure Modes:**
- FTS5 query syntax error → empty results (mitigate: query sanitization)
- Graph traversal infinite loop → timeout (mitigate: max depth limit)
- Budget exceeded → truncated context (expected behavior, log warning)
- Database locked → retry with exponential backoff

**Implementation Priority:** HIGH (already exists, needs integration)
