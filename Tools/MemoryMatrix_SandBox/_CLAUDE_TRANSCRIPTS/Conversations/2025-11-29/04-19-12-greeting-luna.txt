**Research Completed:**

### How It Works

1. **State Machine** tracks where user is (which app, which sub-state like "editing_hook")
2. **JSON Logic Files** define per-state rules: what data to include/exclude/compress, valid actions, HUD display
3. **Context Funnel** assembles the optimized context packet based on current state rules
4. **Luna Persona** receives the filtered context and responds - personality stays constant, but knowledge adapts
5. **Memory System** persists across sessions, compresses over time, syncs via Hub
