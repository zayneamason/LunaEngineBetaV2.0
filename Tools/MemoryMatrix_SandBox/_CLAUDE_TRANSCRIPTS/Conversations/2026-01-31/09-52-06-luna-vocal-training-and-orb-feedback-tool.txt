### **1. Emotion Analyzer**

**Purpose:** Infer emotional tone from Luna's response text *before* it's spoken.

**Input:**
- Text chunk (streaming or full response)
- Current personality state
- Conversation context (optional)

**Output:**
- Emotion label: `excited`, `curious`, `thoughtful`, `playful`, `serious`, etc.
- Intensity: 0.0-1.0
- Confidence: 0.0-1.0

**How it works:**
- Lightweight NLP: keyword matching, punctuation analysis, sentence structure
- Optional: Local sentiment model (DistilBERT, lightweight transformer)
- Maps text features → emotion space

**Example:**
```python
text = "Oh wow, that's fascinating! Let me dive into this..."
emotion = analyzer.analyze(text)
# → {"label": "excited", "intensity": 0.8, "confidence": 0.9}
```

### **2. Prosody Modulator**

**Purpose:** Adjust prosody parameters in real-time based on inferred emotion.

**Input:**
- Base prosody (from personality mapping)
- Emotion state from analyzer
- Text segment boundaries (for phrase-level modulation)

**Output:**
- Adjusted prosody parameters for TTS
- Timing markers for orb sync

**How it works:**
- Starts with baseline prosody from `ProsodyMapper`
- Applies emotion modifiers dynamically
- Can vary prosody *within* a single response (e.g., excited start → thoughtful end)
- Emits phrase boundaries for orb state transitions

**Example:**
```python
# Base prosody from personality
base = ProsodyParameters(rate=1.0, pitch=0.0, pitch_variation=0.5)

```

```javascript
function ParticleSystem({ intensity, emotion }) {
  const canvasRef = useRef(null);
  
  useEffect(() => {
    const ctx = canvasRef.current.getContext('2d');
    const particles = [];

      // Clear canvas
      ctx.clearRect(0, 0, width, height);

      particles.forEach(p => {
        p.y -= p.speed;
        p.alpha -= 0.01;
```

## **Implementation**

### **WebSocket Server (Shared State Hub)**

```python
# tools/performance_studio/server.py

import asyncio
import websockets
import json
from typing import Set

class PerformanceStudioServer:
    def __init__(self):
        self.clients: Set[websockets.WebSocketServerProtocol] = set()
        self.state = {
            "timeline_position": 0.0,
            "orb_state": "idle",
            "prosody": {"rate": 1.0, "pitch": 0.0},
            "emotion": "neutral",
            "intensity": 0.5,
        }
    
    async def register(self, websocket):
        """New client connects."""
        self.clients.add(websocket)
        
        # Send current state to new client
        await websocket.send(json.dumps({
            "type": "state_sync",
            "state": self.state
        }))
    
    async def unregister(self, websocket):
        """Client disconnects."""
        self.clients.remove(websocket)
    
    async def broadcast(self, message):
        """Send to all connected clients."""
        if self.clients:
            await asyncio.gather(
                *[client.send(message) for client in self.clients],
                return_exceptions=True
            )
    
    async def handle_message(self, websocket, message):
        """Handle incoming state updates."""
        data = json.loads(message)
        
        if data["type"] == "state_update":
            # Update server state
            self.state.update(data["state"])
            
            # Broadcast to all other clients
            await self.broadcast(json.dumps({
                "type": "state_changed",
                "state": self.state,
                "source": data.get("source", "unknown")
            }))
    
    async def handler(self, websocket, path):
        """WebSocket connection handler."""
        await self.register(websocket)
        try:
            async for message in websocket:
                await self.handle_message(websocket, message)
        finally:
            await self.unregister(websocket)
    
    async def start(self):
```

## Executive Summary

**Performance Studio** is an iTerm2-based training tool for tuning Luna's vocal performance and orb visualizations. It provides real-time collaboration between human operators and AI agents to analyze transcripts, adjust prosody parameters, map emotions to orb states, and generate training data.

The system bridges offline training (Performance Studio) with live production (Luna Hub) through a shared Performance Layer in Luna Engine.

---

## Problem Statement

### Current State
- Voice prosody is static per-personality, no emotion-driven modulation
- Orb gestures are explicit in text (`*pulses excitedly*`), not inferred
- No tooling to tune voice/orb mappings systematically
- No training data capture for performance improvements
- No way to observe/tune live Luna Hub sessions

### Desired State
- Real-time emotion detection from Luna's response text
- Dynamic prosody modulation based on detected emotion
- Automatic orb state emission synchronized with speech
- Training tool for human + agent collaboration
- Live observation and tuning of production sessions
- Exportable training data for LoRA fine-tuning

#### 1.1 Emotion Analyzer

**File:** `emotion_analyzer.py`

**Purpose:** Detect emotional tone from text before speech synthesis.

**Interface:**
```python
class EmotionAnalyzer:
    def analyze(
        self,
        text: str,
        personality: Dict,
        context: Optional[List[str]] = None
    ) -> EmotionAnalysis:
        """
        Analyze text for emotional content.
        
        Args:
            text: Text to analyze
            personality: Current personality state
            context: Previous phrases for context
            
        Returns:
            EmotionAnalysis(
                emotion: str,           # "excitement", "curiosity", etc.
                intensity: float,       # 0.0-1.0
                confidence: float,      # 0.0-1.0
                detected_gesture: Optional[str],  # Explicit gesture if found
                phrase_boundaries: List[int]      # Character positions
            )
        """
```

**Detection Strategy:**
1. **Explicit gestures** (regex): `*pulses excitedly*` → emotion="excitement"
2. **Punctuation analysis**: Multiple `!` → high intensity
3. **Keyword matching**: "wow", "fascinating" → excitement
4. **Sentence structure**: Questions → curiosity
5. **Personality modulation**: High energy personality → boost detected intensity

**Training Data Usage:**
```sql

FROM emotion_mappings
WHERE text_signature = hash(text_features)
ORDER BY confidence DESC
LIMIT 1;
```

#### 1.2 Prosody Modulator

**File:** `prosody_modulator.py`

**Purpose:** Adjust TTS parameters based on detected emotion.

**Interface:**

```python
class ProsodyModulator:
    def modulate(
        self,
        base_prosody: ProsodyParameters,
        emotion: str,
        intensity: float,
        phrase_position: Optional[int] = None
    ) -> ProsodyParameters:
        """
        Apply emotion-driven prosody adjustments.
        
        Args:
            base_prosody: Base parameters from personality
            emotion: Detected emotion
            intensity: Emotion intensity (0.0-1.0)
            phrase_position: Position in response for gradual changes
            
        Returns:
            Adjusted ProsodyParameters
        """
```

**Modulation Logic:**
```python
# Base from personality
base = ProsodyParameters(rate=1.0, pitch=0.0, pitch_variation=0.5)

# Look up trained tuning
tuning = self.db.get_prosody_tuning(emotion, intensity)

if tuning:
    # Apply learned deltas
    return ProsodyParameters(
        rate=base.rate + tuning.rate_delta,
        pitch=base.pitch + tuning.pitch_delta,
        pitch_variation=base.pitch_variation + tuning.variation_delta,
        pause_duration=base.pause_duration * tuning.pause_multiplier
    )
else:
    # Fallback to default emotion mapping
    return self._apply_default_emotion(base, emotion, intensity)
```

#### 1.3 Orb State Emitter

**Interface:**

```python
class OrbStateEmitter:
    def emit_state(
        self,
        emotion: str,
        intensity: float,
        gesture: Optional[str] = None,
        speech_phase: str = "idle"
    ) -> OrbState:
        """
        Generate orb state from emotion.
        
        Args:
            emotion: Detected emotion
            intensity: Emotion intensity
            gesture: Explicit gesture if detected
            speech_phase: "idle" | "listening" | "processing" | "speaking"
            
        Returns:
            OrbState(
                state: str,              # "pulse", "spin", "glow", etc.

```

**Mapping Priority:**
1. **Explicit gesture** (if detected) → use gesture mapping
2. **System phase** (if not idle) → use system state
3. **Emotion + intensity** → use emotion mapping
4. **Fallback** → idle state

#### 1.4 Performance Orchestrator

**File:** `orchestrator.py`

**Purpose:** Main coordination layer, wires all components together.

**Interface:**
```python
class PerformanceOrchestrator:
    def __init__(self, db_path: str = "data/performance_tunings.db"):
        self.analyzer = EmotionAnalyzer()
        self.modulator = ProsodyModulator()
        self.emitter = OrbStateEmitter()
        self.store = TrainingStore(db_path)
        
        # Load trained mappings at init
        self._load_tunings()
    
    async def process_response(
        self,
        text: str,
        personality: Dict,
        speech_phase: str = "speaking"
    ) -> PerformanceResult:
        """
        Full pipeline: analyze → modulate → emit.
        
        Returns:
            PerformanceResult(
                emotion: EmotionAnalysis,
                prosody: ProsodyParameters,
                orb_state: OrbState
            )
        """
        # 1. Analyze emotion
        emotion = await self.analyzer.analyze(text, personality)
        
        # 2. Modulate prosody
        base_prosody = self._get_base_prosody(personality)
        prosody = self.modulator.modulate(
            base_prosody,
            emotion.emotion,
            emotion.intensity
        )
        
        # 3. Emit orb state
        orb_state = await self.emitter.emit_state(
            emotion.emotion,
            emotion.intensity,
            emotion.detected_gesture,
            speech_phase
        )
        
        return PerformanceResult(emotion, prosody, orb_state)
```

```javascript
// preview/src/Waveform.jsx

import { useEffect, useRef } from 'react';

export function Waveform({ position, audioData }) {
  const canvasRef = useRef(null);
  
  useEffect(() => {
    const canvas = canvasRef.current;
    const ctx = canvas.getContext('2d');
    
    // Clear
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    
    // Draw waveform
    if (audioData) {
      ctx.strokeStyle = '#a78bfa';
      ctx.lineWidth = 2;
      ctx.beginPath();
      
      for (let i = 0; i < audioData.length; i++) {
        const x = (i / audioData.length) * canvas.width;
        const y = (audioData[i] + 1) * canvas.height / 2;
        
        if (i === 0) ctx.moveTo(x, y);
        else ctx.lineTo(x, y);
      }
      
      ctx.stroke();
    }
    
    // Draw playhead
    const x = position * canvas.width;
    ctx.strokeStyle = '#fff';
    ctx.lineWidth = 2;
    ctx.beginPath();
    ctx.moveTo(x, 0);
    ctx.lineTo(x, canvas.height);
    ctx.stroke();
    
  }, [position, audioData]);
  
  return <canvas ref={canvasRef} width={800} height={120} />;
}
```

```javascript
// preview/src/ParticleSystem.jsx

import { useEffect, useRef } from 'react';

export function ParticleSystem({ emotion, intensity }) {
  const canvasRef = useRef(null);
  const particlesRef = useRef([]);
  
  useEffect(() => {
    const canvas = canvasRef.current;
    const ctx = canvas.getContext('2d');
    
    // Generate particles based on intensity
    const particleCount = Math.floor(intensity * 100);
    
    // Add new particles
    while (particlesRef.current.length < particleCount) {
      particlesRef.current.push({
        x: Math.random() * canvas.width,
        y: canvas.height,
        speed: 0.5 + Math.random() * 2,
        alpha: 1,
        size: 1 + Math.random() * 3
      });
    }
    
    // Remove excess particles
    particlesRef.current.splice(particleCount);
    
    // Animation loop
    let animationId;
    
    function animate() {
      ctx.clearRect(0, 0, canvas.width, canvas.height);
      
      particlesRef.current.forEach((p, i) => {
        // Update
        p.y -= p.speed;
        p.alpha -= 0.005;
        
        // Remove dead particles
        if (p.alpha <= 0) {
          particlesRef.current[i] = {
            x: Math.random() * canvas.width,
            y: canvas.height,
            speed: 0.5 + Math.random() * 2,
            alpha: 1,
            size: 1 + Math.random() * 3
          };
        }
        
        // Draw
        ctx.fillStyle = `rgba(167, 139, 250, ${p.alpha})`;
        ctx.fillRect(p.x, p.y, p.size, p.size);
      });
      
      animationId = requestAnimationFrame(animate);
    }
    
    animate();
    
    return () => cancelAnimationFrame(animationId);
  }, [emotion, intensity]);
  
  return (
    <canvas 
      ref={canvasRef} 
      width={800} 
      height={600}
      style={{ position: 'absolute', pointerEvents: 'none' }}
    />
  );
}
```

#### 2.4 Preview WebSocket Server

**File:** `server.py`

**Purpose:** Hub for TUI ↔ Preview ↔ MCP communication.

**Implementation:**
```python
# tools/performance_studio/server.py

import asyncio
import websockets
import json
from typing import Set
from dataclasses import dataclass, asdict

@dataclass
class StudioState:
    """Shared state across all clients."""
    timeline_position: float = 0.0
    orb_state: str = "idle"
    orb_brightness: int = 100
    orb_color: str = "#a78bfa"
    emotion: str = "neutral"
    intensity: float = 0.5
    prosody: dict = None
    
    def __post_init__(self):
        if self.prosody is None:
            self.prosody = {"rate": 1.0, "pitch": 0.0}

class PerformanceStudioServer:
    def __init__(self):
        self.clients: Set[websockets.WebSocketServerProtocol] = set()
        self.state = StudioState()
    
    async def register(self, websocket):
        """New client connects."""
        self.clients.add(websocket)
        print(f"Client connected. Total clients: {len(self.clients)}")
        
        # Send current state to new client
        await websocket.send(json.dumps({
            "type": "state_sync",
            "state": asdict(self.state)
        }))
    
    async def unregister(self, websocket):
        """Client disconnects."""
        self.clients.remove(websocket)
        print(f"Client disconnected. Total clients: {len(self.clients)}")
    
    async def broadcast(self, message: str, exclude=None):
        """Send to all connected clients except exclude."""
        if not self.clients:
            return
        
        targets = self.clients - {exclude} if exclude else self.clients
        
        await asyncio.gather(
            *[client.send(message) for client in targets],
            return_exceptions=True
        )
    
    async def handle_message(self, websocket, message):
        """Process incoming state updates."""
        data = json.loads(message)
        
        if data["type"] == "state_update":
            # Update server state
            state_update = data["state"]
            for key, value in state_update.items():
                if hasattr(self.state, key):
                    setattr(self.state, key, value)
            
            # Broadcast to other clients
            await self.broadcast(json.dumps({
                "type": "state_changed",
                "state": asdict(self.state),
                "source": data.get("source", "unknown")
            }), exclude=websocket)
        
        elif data["type"] == "scrub":
            # Timeline scrub
            self.state.timeline_position = data["position"]
            await self.broadcast(json.dumps({
                "type": "timeline_scrub",
                "position": data["position"]
            }), exclude=websocket)
        
        elif data["type"] == "orb_update":
            # Direct orb state update
            self.state.orb_state = data.get("state", self.state.orb_state)
            self.state.orb_brightness = data.get("brightness", self.state.orb_brightness)
            self.state.orb_color = data.get("color", self.state.orb_color)
            
            await self.broadcast(json.dumps({
                "type": "orb_state",
                **data
            }), exclude=websocket)
    
    async def handler(self, websocket, path):
        """WebSocket connection handler."""
        await self.register(websocket)
        try:
            async for message in websocket:
                await self.handle_message(websocket, message)
        except websockets.exceptions.ConnectionClosed:
            pass
        finally:
            await self.unregister(websocket)
    
    async def start(self):
        """Start WebSocket server."""
        async with websockets.serve(self.handler, "localhost", 3001):
            print("🎭 Performance Studio Server running on ws://localhost:3001")
            await asyncio.Future()  # Run forever

if __name__ == "__main__":
    server = PerformanceStudioServer()
    asyncio.run(server.start())
```

```python
from fastapi import WebSocket, WebSocketDisconnect
from luna.performance import PerformanceOrchestrator

# Global performance instance
performance = PerformanceOrchestrator()

@app.websocket("/ws/performance")
async def performance_websocket(websocket: WebSocket):
    """
    Performance tuning WebSocket.
    
    Used by Performance Studio to:
    - Observe live conversations
    - Inject real-time tunings
    - Receive state updates
    """
    await websocket.accept()
    
    # Register as observer
    performance.add_observer(websocket)
    
    try:
        while True:
            data = await websocket.receive_json()
            
            if data["type"] == "tune_prosody":
                # Update prosody tuning
                performance.modulator.update_tuning(
                    emotion=data["emotion"],
                    delta=data["delta"]
                )
                
                # Save to DB
                performance.store.add_prosody_tuning(
                    emotion=data["emotion"],
                    **data["delta"]
                )
            
            elif data["type"] == "tune_orb":
                # Update orb mapping
                performance.emitter.update_mapping(
                    gesture=data.get("gesture"),
                    emotion=data.get("emotion"),
                    state=data["state"]
                )
                
                # Save to DB
                performance.store.add_orb_mapping(
                    gesture=data.get("gesture"),
                    emotion=data.get("emotion"),
                    orb_state=data["state"],
                    **data.get("orb_params", {})
                )
            
            elif data["type"] == "observe_session":
                # Stream conversation events to Performance Studio
                # (already registered as observer above)
                pass
    
    except WebSocketDisconnect:
        performance.remove_observer(websocket)
```

```python
import websockets
import json
from typing import Optional, Dict, List
from luna.performance import PerformanceOrchestrator

# Global connections
_performance_ws = None
_studio_ws = None

async def _ensure_connections():
    """Ensure connections to Luna Engine and Performance Studio."""
    global _performance_ws, _studio_ws
    
    # Connect to Luna Engine performance endpoint
    if not _performance_ws:
        try:
            _performance_ws = await websockets.connect(

            )
        except Exception as e:
            print(f"Luna Engine not available: {e}")
    
    # Connect to Performance Studio (if running)
    if not _studio_ws:
        try:
            _studio_ws = await websockets.connect(
                "ws://localhost:3001"
            )
        except Exception:
            # Studio not running, that's okay
            pass

async def luna_tune_performance(
    command: str,
    text: Optional[str] = None,
    timeline_position: Optional[float] = None,
    emotion: Optional[str] = None,
    prosody_delta: Optional[Dict] = None,
    orb_state: Optional[str] = None,
    show_preview: bool = True,
    reason: Optional[str] = None
) -> Dict:
    """
    Tune Luna's performance layer (voice + orb).
    
    Commands:
    - "analyze": Analyze text for emotion/prosody/orb suggestions
    - "tune_prosody": Adjust prosody mapping
    - "tune_orb": Adjust orb state mapping
    - "scrub": Move timeline position (Performance Studio only)
    - "observe": Start observing live Luna Hub session
    - "export": Export training data
    - "status": Get current tuning statistics
    
    Args:
        command: Action to perform
        text: Text to analyze (for "analyze")
        timeline_position: Timeline position (for "scrub")
        emotion: Emotion name (for tuning commands)
        prosody_delta: Prosody adjustments (for "tune_prosody")
        orb_state: Orb state (for "tune_orb")
        show_preview: Show in Performance Studio preview
        reason: Human-readable explanation for this action
    
    Returns:
        Result dict with status and data
    """
    await _ensure_connections()
    
    if command == "analyze":
        if not text:
            return {"error": "text required for analyze command"}
        
        # Run analysis through Performance Layer
        orchestrator = PerformanceOrchestrator()
        result = await orchestrator.process_response(
            text=text,
            personality={"energy": 0.7, "warmth": 0.6}  # TODO: Get from context
        )
        
        # Send to Performance Studio if connected
        if _studio_ws and show_preview:
            await _studio_ws.send(json.dumps({
                "type": "analysis_result",
                "text": text,
                "emotion": result.emotion.emotion,
                "intensity": result.emotion.intensity,
                "prosody": {
                    "rate": result.prosody.rate,
                    "pitch": result.prosody.pitch
                },
                "orb_state": result.orb_state.state,
                "reason": reason
            }))
        
        return {
            "status": "analyzed",
            "emotion": result.emotion.emotion,
            "intensity": result.emotion.intensity,
            "confidence": result.emotion.confidence,
            "prosody": {
                "rate": result.prosody.rate,
                "pitch": result.prosody.pitch,
                "pitch_variation": result.prosody.pitch_variation
            },
            "orb_state": result.orb_state.state,
            "orb_brightness": result.orb_state.brightness
        }
    
    elif command == "tune_prosody":
        if not emotion or not prosody_delta:
            return {"error": "emotion and prosody_delta required"}
        
        # Send to Luna Engine
        if _performance_ws:
            await _performance_ws.send(json.dumps({
                "type": "tune_prosody",
                "emotion": emotion,
                "delta": prosody_delta,
                "reason": reason
            }))
        
        # Send to Performance Studio
        if _studio_ws:
            await _studio_ws.send(json.dumps({
                "type": "state_update",
                "source": "mcp",
                "state": {
                    "prosody": prosody_delta
                }
            }))
        
        return {
            "status": "tuned",
            "emotion": emotion,
            "prosody_delta": prosody_delta,
            "reason": reason
        }
    
    elif command == "tune_orb":
        if not orb_state:
            return {"error": "orb_state required"}
        
        # Send to Luna Engine
        if _performance_ws:
            await _performance_ws.send(json.dumps({
                "type": "tune_orb",
                "emotion": emotion,
                "state": orb_state,
                "reason": reason
            }))
        
        # Send to Performance Studio
        if _studio_ws:
            await _studio_ws.send(json.dumps({
                "type": "orb_update",
                "source": "mcp",
                "state": orb_state
            }))
        
        return {
            "status": "tuned",
            "emotion": emotion,
            "orb_state": orb_state,
            "reason": reason
        }
    
    elif command == "scrub":
        if timeline_position is None:
            return {"error": "timeline_position required"}
        
        # Send to Performance Studio only
        if _studio_ws:
            await _studio_ws.send(json.dumps({
                "type": "scrub",
                "position": timeline_position,
                "source": "mcp"
            }))
            
            return {
                "status": "scrubbed",
                "position": timeline_position
            }
        else:
            return {"error": "Performance Studio not connected"}
    
    elif command == "observe":
        # Start observing live Luna Hub session
        if _performance_ws:
            await _performance_ws.send(json.dumps({
                "type": "observe_session",
                "session_id": "current"
            }))
            
            return {
                "status": "observing",
                "message": "Now observing live Luna Hub session"
            }
        else:
            return {"error": "Luna Engine not connected"}
    
    elif command == "export":
        # Export training data
        orchestrator = PerformanceOrchestrator()
        count = orchestrator.store.export_training_data(
            output_path="data/performance_training.jsonl",
            format="jsonl"
        )
        
        return {
            "status": "exported",
            "format": "jsonl",
            "path": "data/performance_training.jsonl",
            "record_count": count
        }
    
    elif command == "status":
        # Get tuning database statistics
        orchestrator = PerformanceOrchestrator()
        stats = orchestrator.store.get_statistics()
        
        return {
            "status": "ok",
            "statistics": stats
        }
    
    else:
        return {"error": f"Unknown command: {command}"}
```

## Known Limitations

1. **Emotion detection is rule-based** - No ML model initially, relies on keywords and punctuation
2. **Single orb only** - System doesn't support multiple simultaneous orbs
3. **No audio waveform extraction** - Waveform visualization requires manual audio file input
4. **macOS only** - Window positioning uses macOS-specific tools (Rectangle, AppleScript)
5. **SQLite single-writer** - Multiple Performance Studio instances may conflict
6. **No real-time collaboration between multiple humans** - One operator at a time

---

## Performance Notes

- Emotion analysis: ~5ms per phrase (rule-based)
- Prosody modulation: <1ms (lookup + calculation)
- Orb state emission: <1ms (WebSocket broadcast)
- Total overhead per response: <10ms (negligible)
- WebSocket latency (TUI → Preview): ~5-10ms
- Training database queries: <5ms (indexed lookups)

---

## Security Considerations

- Performance Studio runs **locally only** (localhost:3000, localhost:3001)
- Training database contains **no sensitive user data**
- WebSocket connections are **unauthenticated** (local development only)
- For production deployment, add:
  - Authentication tokens for WebSocket
  - HTTPS for web preview
  - Database encryption for sensitive tunings
