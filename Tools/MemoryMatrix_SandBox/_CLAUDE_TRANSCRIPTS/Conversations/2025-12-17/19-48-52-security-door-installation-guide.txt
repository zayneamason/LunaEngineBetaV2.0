What can you tell me about security door installation

--------------------------------------------------------------------------------

[ASSISTANT] 2025-12-17T19:49:18.446480Z
 Security door installation breaks down into a few main categories depending on what you're securing and your budget:

**Residential vs. Commercial**
- Residential: Steel security screen doors, reinforced entry doors, or full security door replacements. Most are DIY-friendly if you're comfortable with door framing.
- Commercial: Heavy-gauge steel doors, often with integrated access control (keypad, card reader, biometric). Usually requires professional install due to fire codes and ADA compliance.

**Key Components**
- **Door itself**: Steel gauge matters—16-gauge is standard residential, 14-gauge or heavier for serious security
- **Frame**: Often the weak point. Steel frames or door frame reinforcement kits (like Door Armor) are critical—most break-ins attack the jamb, not the door
- **Strike plate**: Extended strike plates with 3" screws into studs, not just the jamb
- **Hinges**: Security hinges with non-removable pins on outswing doors
- **Lock**: Deadbolt with minimum 1" throw, Grade 1 or 2 rated

**Install Considerations**
- Rough opening sizing—security doors often need beefier frames
- Shimming and plumb matter more when you're hanging heavier steel
- Weather sealing if it's an exterior door
- Threshold height for ADA if commercial

**Ballpark Costs**
- Basic steel security screen door: $200-600 + install
- Solid steel entry door with frame: $800-2,500+
- Commercial-grade with access control: $2,000-10,000+

How does one become a professional at installing these doors?

--------------------------------------------------------------------------------

[ASSISTANT] 2025-12-17T19:51:24.703014Z
 Pretty straightforward path, actually:

**The Practical Route (Most Common)**

1. **Start in general door/hardware installation** - Work for a locksmith, general contractor, or door installation company. Learn standard door hanging first—residential entry doors, interior doors, weatherstripping, hardware.

2. **Move into commercial or security-focused work** - Companies that specialize in security doors, access control, or commercial hardware. This is where you learn the heavier-gauge stuff, frame reinforcement, and code compliance.

3. **Get on with a security door manufacturer's install team** - Companies like Assa Abloy, Allegion, or regional security door distributors often have dedicated install crews or certified installer networks.

**Certifications That Matter**

- **DHI (Door and Hardware Institute)** - Offers credentials like AHC (Architectural Hardware Consultant) and CDC (Certified Door Consultant). These are gold standard for commercial work.
- **ALOA (Associated Locksmiths of America)** - If you're blending locksmithing with door security
- **Manufacturer certifications** - Many commercial security door brands require installer certification for warranty work

**What Gets You Hired**

- Experience with hollow metal frames and commercial steel doors
- Understanding of fire-rated assemblies and ADA requirements
- Familiarity with access control integration (even basic low-voltage wiring)
- Ability to read architectural hardware schedules

**Timeline**

Realistically, 2-3 years working under experienced installers gets you competent. 5+ years and manufacturer certs makes you a specialist who can command better rates.

[ASSISTANT] 2025-12-17T20:10:51.023932Z
 Nice, that's a manageable volume to systematize. Here's how I'd structure it:

**Pre-Job (Win the Work)**

- **Bid tight but not cheap**—institutional buyers expect professional pricing. Lowballing signals inexperience.
- **Site survey every job**—measure frames, check rough openings, note existing hardware, photograph everything. Surprises kill margins.
- **Spec sheet upfront**—get exact door/frame/hardware specs in writing before you quote. "Security door" means different things to different people.

**Materials & Logistics**

- **Establish distributor relationships NOW**—find 1-2 commercial door/hardware suppliers in LA (Beacon Commercial Door, Door + Hardware Consultants, etc.). Net-30 terms, contractor pricing, and they can help you spec jobs correctly.
- **Kit jobs in advance**—all hardware, frames, doors, fasteners, shims staged and labeled per opening before you roll to site. Zero hardware store runs mid-job.
- **Vehicle setup**—dedicated install vehicle with organized storage. You're losing money every time you dig for a hinge.

**On-Site Execution**

- **One lead, one support**—lead handles layout, plumb, critical installs. Support preps openings, stages materials, handles punch list.
- **Door-by-door checklist**—frame plumb/level, anchoring, door hung, hardware installed, closer adjusted, fire label intact (if rated), swing/latch/lock function. Initial each door.
- **Photo document every install**—CYA for warranty, disputes, and future portfolio.

**Post-Job**

- **Punch list same day if possible**—don't give them time to find problems you could've caught
- **Inspection-ready paperwork**—fire door labels logged, hardware schedules completed, ADA compliance noted
- **Invoice immediately**—net-30 starts when you send it, not when you finish

**Time Math**

20 doors across 1-2 sites, assuming straightforward installs:
- 2-3 doors per day per two-person crew is sustainable without burning out
- That's roughly 7-10 working days of install
- Leaves room for survey, material runs, admin, and chasing the next job

**The Winning Edge**

Institutional clients rehire based on three things:
1. **Clean work**—no callbacks, no punch list drama
2. **Documentation**—they need paper trails for inspections and audits
3. **Communication**—proactive updates, no surprises

Do that better than the sloppy crews they've dealt with before, you'll get referrals fast.

[ASSISTANT] 2025-12-17T20:39:31.345154Z
 Yeah, that glass door nightmare is exactly the kind of thing that kills margins and trust. Here's how to systematize it:

**The Core Problem**

Multiple openings → multiple specs → multiple parts → chaos if anything gets swapped or mislabeled.

**The System: Opening ID + Tagging**

Every opening gets a unique ID that follows it through the entire process:

**1. Site Survey Phase**
- Assign each opening a code: `[Building/Floor]-[Room]-[Opening#]`
  - Example: `A2-305-D1` = Building A, Floor 2, Unit 305, Door 1
- Photograph each opening with a whiteboard/phone showing the ID in frame
- Measurements recorded against that specific ID, not just "master bathroom door"

**2. Spec Sheet / Order Phase**
- Order form lists every opening ID with its specific specs
- Each line item ties back to the ID: `A2-305-D1: 36"x80" steel, 16ga, LH inswing, mortise prep`
- When ordering from distributor, the ID goes on the PO line item

**3. Receiving / Staging**
- When materials arrive, immediately tag/label each piece with its opening ID
- Painter's tape + sharpie, or printed labels
- Stage materials grouped by opening, not by type
- Kit everything together: door + frame + hardware + fasteners for each ID

**4. On-Site Install**
- Installer grabs the kit for `A2-305-D1`, walks to that opening, installs
- Checklist per opening ID, not just "20 doors done"

**Tools That Help**

| Approach | Complexity | Cost |
|----------|------------|------|
| Spreadsheet + whiteboard photos | Low | Free |
| Google Sheets + QR codes on labels | Medium | ~$50 (label printer) |
| Field management app (Fieldwire, Procore) | High | $$/month |

**Quick & Dirty Version for You**

1. **Google Sheet** with columns: Opening ID | Location | Width | Height | Swing | Hardware | Photo Link | Ordered | Received | Installed
2. **Photo naming convention**: `A2-305-D1_survey.jpg`
3. **Physical labels**: Print or handwrite ID on tape, stick on door/frame immediately on delivery
4. **One person owns the sheet** and updates status as things move

**The GC Fuckup Insurance**

When you're downstream of a GC's measurements:
- **Re-verify critical dimensions yourself** before ordering—especially custom/glass/fire-rated
- Document that you verified (photo with tape measure + ID visible)
- If GC provides specs, get it in writing and confirm in email: "Per your measurements, opening A2-305-D1 is 36x80, confirming before I order"

Now if shit goes sideways, you've got paper trail showing the error wasn't yours.
