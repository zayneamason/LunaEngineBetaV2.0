Here is an overview of the key information on familiarity with digital assets, file formats, and file storage/sharing:

Digital Assets
- Digital assets refer to electronic files such as documents, photos, videos, audio files that exist in a digital format and can be stored and transmitted over the internet 
- Common types: images (JPEG, PNG), audio files (MP3, WAV), video files (MP4, AVI), documents (PDF, DOC, TXT)

File Formats 
- A file format is how information is encoded and stored digitally with a file extension indicating the format
- Knowing file formats allows proper opening, editing, sharing of files 
- Some key formats: JPEG and PNG for images; MP3 and WAV for audio; MP4 for video; PDF for documents

File Storage/Sharing
- Files can be stored locally (on a computer/device hard drive) or remotely using cloud storage (Dropbox, Google Drive, iCloud etc.)
- Cloud services make it easy to store large number of files reliably and share them 
- Permissions, links, collaborator access can be configured on these platforms for seamless sharing
- Bandwidth limits, storage capacity, security are things to keep in mind

Here is an overview of video codecs:

What is a Video Codec?
- A video codec is software that compresses and decompresses digital video. It encodes (compresses) raw digital video into a file when it is recorded or streamed, and decodes it when it's played back.

Main Types of Video Codecs:
1. H.264 (AVC) - The most common video codec. Offers high quality HD video with efficient compression. Used for online streaming, Blu-ray discs and more.

2. HEVC (H.265) - More advanced codec that can compress video files at half the size of H.264 with the same quality. Allows high resolutions like 4K/8K. Still being adopted across many devices and services. 

3. VP9 - Open and royalty free video codec developed by Google. Used by YouTube and compatible with 4K video. Offers efficient compression comparable to HEVC.

4. AV1 - Newest royalty free codec. Enables high resolution up to 8K with greater compression efficiency. Expected to be widely adopted for streaming and consumer devices.

Understanding video codecs allows optimizing resolutions and file sizes for tasks like sharing online or storing locally. Newer codecs like HEVC and AV1 provide more compact files capable of better video quality.

Here is a more in-depth overview of video codecs:

How Video Codecs Work
- Video codecs use algorithms to analyze and compress raw digital video by removing spatial and temporal redundancies between video frames. 
- Compression is essential to reduce video file sizes for storage and streaming.
- Codecs encode video when recording/uploading, and decode it when editing/playing back.

Major Codec Types
1. Lossy - These codecs compress by losing some quality which is generally imperceptible. Includes h.264, HEVC, VP9, AV1.

2. Lossless - As the name suggests, all video information is preserved. Would result in enormous files unsuitable for sharing/streaming. Used in production/editing.

3. Codec Families - h.26x (h.264 etc), MPEG (MPEG-2, 4 etc.)

Key Factors
- Compression Ratio - Smaller the file at a given quality, the better. HEVC offers 50% better compression over h.264.

- CPU Usage - More compression requires more processor resources. Needs hardware acceleration support.

- Quality - Balance between compression and visual quality.

- Supported Resolutions - Up to 4K/8K for HEVC/VP9 and higher for AV1.

- Hardware/Software Support - Whether devices, web browsers support playback for a codec. AV1 and HEVC still have relatively limited hardware support.

In summary, the choice depends on application (streaming vs Blu-ray) and other factors like device compatibility. H.264 remains ubiquitous while HEVC and AV1 promise better quality and resolutions for the future.
