## Integration Patterns

### Pattern 1: Tool Call
```
┌─────────┐                              ┌─────────┐
│  LUNA   │                              │  EDEN   │
└────┬────┘                              └────┬────┘
     │                                        │
     │  1. User: "make an image of..."       │
     │  ──────────────────────────────────▶  │
     │                                        │
     │  2. Tool: eden_create_image           │
     │  {prompt: "..."}                       │
     │  ──────────────────────────────────▶  │
     │                                        │
     │  3. Result: {task_id: "abc"}          │
     │  ◀──────────────────────────────────  │
     │                                        │
     │  4. Poll until complete               │
     │  ──────────────────────────────────▶  │
     │                                        │
     │  5. Result: {url: "..."}              │
     │  ◀──────────────────────────────────  │
     │                                        │
     │  6. "Here's your image: [display]"    │
     ▼                                        ▼
```
